Tabla de Principales Etiquetas HTML

Nombre de Etiqueta Descripción
article Indica la sección de un artículo, pueden haber varios artículos en una misma sección.
html Indica el comienzo del documento HTML.
head Indica que empieza la cabecera de la página, generalmente es el título.
section Definición de una seccción en el código, indicando que es un bloque o una sección.
title Define el título del documento, el cual se muestra en la barra de título del navegador o en las pestañas de página.
nav Define una sección que solamente contiene enlaces de navegación
footer Define el pie de una página o sección. Usualmente contiene un mensaje de derechos de autoría, algunos enlaces a información legal o direcciones para dar información de retroalimentación.
body Indica que es el cuerpo de la página, aquí van los textos, las imágenes y todo el contenido de la página.
h1, h2, h3 Se utiliza para los títulos, el tamaño del texto es más grande.
a Se utiliza para agregar enlaces al la página.
table Se utiliza para el diseño de tablas. Dentro de ella se utilizan las etiquetas "tr" para las filas y las "td" para las celdas.
p Indica que lo que está dentro de ésta etiqueta es un párrafo.
img Se utiliza para insertar imágenes a la página.
ul Indica que lo que esta dentro de ésta etiqueta es una lista no ordenada, dentro de ella se utiliza la etiqueta "li" para cada guión, y se utiliza "ol" para una lista ordenada.
b y strong Se utiliza para resaltar el texto.
u Se utiliza para subrayar el texto.
i y em Se utiliza para el texto en cursiva.

Tabla de Estilos utilizados en CSS

Nombre del Estilo Descripción
color Se utiliza para definir el color del texto. Se puede escribir el Nombre del color en ingles "color: red", para rojo, o en el sistema RGB, "color: rgb( 250, 8, 22 )" para el mismo rojo.
Background Se utiliza para definir el color de fondo, se puede definir el color escribiendo su nombre en ingles o en la notación RGB hexadecimal.
border Se utiliza para aplicar estilos, no son visibles, y es como un cuadro imaginario que envuelve todo el contenido del bloque. Son cuatro bordes simultáneamente, border-color,border-width y border-style.
margin Se utiliza para definir el espacio comprendido entre el bloque y el borde de la ventana activa del navegador.
text Define las propiedades que determinan la apariencia del texto en general, text-align sirve para alinear el texto: (left) a la izquierda, (right) a la derecha, (center) centrado y (justify) justificado.
font Permite establecer una sola vez los valores para todas las propiedades en una hoja de estilo. "font-style" permite definir el aspecto de una familia tipográfica entre los valores: normal, italic (cursiva) y oblique. "font-variant", selecciona entre los aspectos normal y small-caps para la familia de fuentes. "font-weight", especifica el peso o grueso de la letra. Algunos tipos de letra sólo están disponibles en normal y bold. "font-size", especifica la dimensión de la letra. Este tamaño puede, a su vez, alterar el aspecto de alguna otra cosa, ya que se usa para calcular la longitud de las unidades em y ex. "line-height", establece la altura de una casilla remarcada por líneas. y "font-family, define una lista de fuentes o familias de fuentes, con un orden de prioridad, para utilizar en un elemento seleccionado."
height Especifica la altura del area de contenido de un elemento. El área de contenido está dentro del padding, borde, y margen del elemento. Las propiedades min-height y max-height sobreescriben el valor de height.
width controla la anchura de la caja de los elementos.Esta propiedad no admite valores negativos y los valores en porcentaje se calculan a partir de la anchura de su elemento padre.
link Representa un elemento que aún no se ha visitado. Coincide con cada elemento no visitado
list Establece las propiedades del marcador de los elementos de la lista, se aplica a todos los elementos de tipo elemento de lista
table Permite asignar un comportamiento de tabla a los elementos que haya dentro,
position Especifica cómo un elemento es posicionado en el documento. Las propiedades top, right, bottom, y left determinan la ubicación final de los elementos posicionados.
overflow Se utiliza para: recortar contenido, dibujar barras de desplazamiento o mostrar el contenido excedente en un elemento a nivel de bloque
float Sirve para alinear un elemento, haciendo que el texto de alrededor del mismo se agrupe entorno a dicho elemento. Sus posibles valores son “left” para alinear el elemento a la izquierda y “right” para alinearlo a la derecha.

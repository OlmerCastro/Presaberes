Tabla de Principales Etiquetas HTML

Nombre de Etiqueta Descripción
html Indica el comienzo del documento HTML.
head Indica que empieza la cabecera de la página, generalmente es el título.
body Indica que es el cuerpo de la página, aquí van los textos, las imágenes y todo el contenido de la página.
h1, h2, h3 Se utiliza para los títulos, el tamaño del texto es más grande.
a Se utiliza para agregar enlaces al la página.
table Se utiliza para el diseño de tablas. Dentro de ella se utilizan las etiquetas "tr" para las filas y las "td" para las celdas.
p Indica que lo que está dentro de ésta etiqueta es un párrafo.
img Se utiliza para insertar imágenes a la página.
ul Indica que lo que esta dentro de ésta etiqueta es una lista no ordenada, dentro de ella se utiliza la etiqueta "li" para cada guión, y se utiliza "ol" para una lista ordenada.
b y strong Se utiliza para resaltar el texto.
u Se utiliza para subrayar el texto.
i y em Se utiliza para el texto en cursiva.

Tabla de Estilos utilizados en CSS

colors Se utiliza para definir el color del texto. Se puede escribir el Nombre del color en ingles "color: red", para rojo, o en el sistema RGB, "color: rgb( 250, 8, 22 )" para el mismo rojo.
Backgrounds Se utiliza para definir el color de fondo, se puede definir el color escribiendo su nombre en ingles o en la notación RGB hexadecimal.
borders Se utiliza para aplicar estilos, no son visibles, y es como un cuadro imaginario que envuelve todo el contenido del bloque.
margins Se utiliza para definir el espacio comprendido entre el bloque y el borde de la ventana activa del navegador.
